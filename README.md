# RF Antennas

Repository focused on antenna design, simulation, construction and validation.

## Objectives

- Learn practical antenna theory
- Model antennas using 4NEC2
- Compare theoretical and measured performance
- Validate designs using NanoVNA
- Document the complete engineering process

## Projects

### Dipoles
- Half-wave dipole
- 120° V-dipole for NOAA APT reception

### Future projects
- QFH antenna
- Turnstile antenna
- Yagi-Uda antennas
- Satellite communication antennas

## Tools

- 4NEC2
- NanoVNA
- Python
